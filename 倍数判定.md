- [ABC 082 A - Round Up the Mean](https://atcoder.jp/contests/abc082/tasks/abc082_a)
	- 余りの切り上げ
- 問題：2つの正整数a,bが与えられます。a, bの平均値をxとします。xの小数点以下を切り上げて得られる整数を出力してください。
- 制約
	- a,bは整数
	- 1 <= a,b <= 100

```
#include <bits/stdc++.h>
using namespace std;
 
int main() {
  int a, b;
 
  cin >> a >> b;
 
  cout << (a+b)/2 + (a+b)%2 << endl;
}
```

```
package main

import "fmt"
 
func main() {
	var a, b int
	fmt.Scan(&a, &b)
 
	fmt.Println((a + b + 1) / 2)
}
```
